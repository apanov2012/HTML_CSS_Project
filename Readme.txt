 Представленный проект представляет из себя веб-страницу,выдержанную в общей тематике ралли Париж-Дакар
 и произвольным,не относящимся к визуальной тематике,текстовым наполнением.
 Схематически веб-страница представляет из себя:
	-Экран с "шапкой", состоящей из горизонтильно расположенных лого ,5 элементов меню на аниированном фоне и титульным изображением на остальной площади экрана
	 Анимированность достигнута использованием в фоне меню соответствующего gif-файла.
	-Экран с 2 текстовыми блоками, расположенными на фоне анимированного фона.Анимированность достигнута использованием в фоне соответствующего gif-файла.
	-Экран с галереей из 6 фотографий. Фотографии являются фоном для соответствующих блоков.
	-Футер с тремя вертикальными группами элементов произвольного нейминга, оформленных в визуальном формате гиперссылок,но таковыми не являющимися,
	 а также закрывающего логотипа.

 При верстве данной страницы использовался flex-подход.
 Каждый из блоков,описанных ниже, является flex-блоком с соответствующим набором свойств.
 Все используемые в данном проекте изображения, в том числе и содержащие анимацию,находятся в папке Images

 Проект состоит из:
  Трех блоков экранов класса Screens со 100%-й высотой блока относительно высоты экрана.
   В состав первого блока .screens:nth-of-type(1) входят:
	 - Фоновый блок с идентификатором "menu_back",включающий в себя:
		- Блок с классом "logo" для размещения открывающего логотипа
		- Блок с классом "menu" для размещения блоков меню.Включает в себя:
			- 5 блоков с классом "menu_button" с наименованием каждого из элементов меню, каждый из которых содержит:
				- Блок с кнопкой пункта меню
				- Фоновый блок для выпадающего меню. Включает в себя:
					- 6 блоков с пунктами выпадающего меню, первый из которых не имеет содержимого
 
   В состав второго блока .screens:nth-of-type(2) входят:
	 - Фоновый блок с идентифиикатором "big_frame" для последующего позиционирования и отладки блоков с текстовым содержимым.Включает в себя:
		- 2 блока с классом "text" с текстовым наполнением

   В состав третьего блока .screens:nth-of-type(3) входят:
	 - Фоновыый блок с идентификатором "gallery" для размещения блоков с фото, включающий в себя:
		- 6 блоков с наименованием .picture:nth-of-type(1)-(6) с уникальными изображениями.
		  Изображения являются фоновыми для каждого из соответствующих блоков.
  Заверщающего блока с идетнификатором "footer", включающего в себя:
	 - Фоновый блок с идентификатором "columns_back" для размещения блоков, имеющих в своем составе блоки с "псевдоссылками".Включает:
		-3 блока с классом "columns", содержащих блоки с "псевдоссылками".Каждый из блоков класса "columns" включает в себя:
			- 6, 4 либо 3 блока с "псевдоссылками" соответственно.
	 -Фоновый блок с идентификатором "footer_logo_back" для размещения и позиционирования блока с заверщающим лого.Включает в себя:
		- Блок с классом "logo" с завершающим лого.
